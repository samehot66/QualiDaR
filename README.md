qualidar/backend> npm install
pip install mysql-connector-python<br>

pip install python-dotenv<br>
pip install pdfminer3<br>

## Build
qualidar> npm build
*Becareful! everything in qulidar/build/upload will be disappeared

## Setting
1. Setting config in .env file
  1.1 For Frontend is in qualidar/.env
  1.2 For Backend is in qualidar/backend/.env
  1.3 For Python is in qualidar/python/.env
2. Create folder name upload in qulidar/build/
3. Move file howtousethequalidar.pdf into qulidar/build/upload
